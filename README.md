## Example of use

```php
require __DIR__ . '/vendor/autoload.php';

$searchTags = new WFTags\SearchTags(WFTags\Enum\Types::USER);

$list = ['Сцена', 'Элез', 'Актриса', 'Аура', 'Интерес', 'Фрик616'];

$define = [
    100 => 'никнейм используется', 
    'неактивный никнейм', 
    'свободный, но возможно зарезервирован', 
    'ник скрыт, но используется'
];

$j = 1;
foreach ($list as $item)
{
    $x = $define[$searchTags->get($item)] ?? false;
    echo "[#$j] $item ($x)\n";
    $j++;
}
```

## Result
```text
[#1] Сцена (никнейм используется)
[#2] Элез (никнейм используется)
[#3] Актриса (никнейм используется)
[#4] Аура (ник скрыт, но используется)
[#5] Интерес (ник скрыт, но используется)
[#6] Фрик616 (свободный, но возможно зарезервирован)
```
